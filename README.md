Майчин Органайзер
За проекта
"Майчин органайзер" е интуитивно уеб приложение, създадено с цел да улесни натовареното ежедневие на майките. То предоставя дигитално пространство за управление на ежедневните задачи, графиците на децата, списъци с покупки и важни бележки за домакинството.
Основни функции (които планираме)
Добавяне и следене на ежедневни задачи.
Организиране на графици.
Лесен и бърз достъп през мобилно устройство или компютър.
Технологии
Проектът е базиран на Google Apps Script среда и използва класически уеб технологии: HTML, CSS и JavaScript.
